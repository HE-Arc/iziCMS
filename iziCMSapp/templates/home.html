FAQ

  • What is iziCMS ?
    iziCMS is a micro Content Management System that lets you edit existing static web pages.

  • How do I install it ?
    It doesn't require any installation, everything is done from this platform.

  • Can I use iziCMS for any website ?
    No, you can only edit static web-pages (.html). You cannot combine iziCMS and some other CMS or framework.

  • How does it work ?
    iziCMS use your FTP configuration to download the pages you want to edit.
    You can select and edit particular elements of the page and iziCMS will upload the result to your website.

  • Can I undo some modification I did with iziCMS ?
    No, iziCMS doesn't provide any history system for now.
    We recommand you to save your files before to use iziCMS, in case something bad happens.

  • Can I add a page to my website ?
    No, iziCMS can only edit files that already exists.

  • Do you store my password in your database ?
    No, your FTP password is stored in your HTTP session the first time you connect to your website.

  • I don't want to give you my FTP password
    OK. We can not force you to trust us.

  • Is the communication secure ?
    Yes, note that iziCMS is under the HTTPS protocol, which means that the communication from you to us is encrypted.
